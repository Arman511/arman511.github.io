👩🏽‍🚀 Projects

Building History Viewer 📜

I made a Github pages web app that will shows a buildings history using the OpenStreetMaps data, it will then show all the building that were at that location. It works best will building that are non-residential. e.g. shops and restaurants. Can be found here.

RR cli Reader 📖

A cli reader for RoyalRoad. Built in Rust and can be found on Github here. Can be compiled by yourself from source with 'cargo build --release' and can be found in the /target/release folder as an application file. More info can be found on the REAdME.md file on the Github repo.

News reader (CLI and GUI) 📰

Made a CLI and GUI news reader in Rust. They use the newsdata.io API to get the news. It can be compiled by yourself from source with 'cargo build --release' and can be found in the /target/release folder as an application file. Made by following a tutorial.

Restaurant Sim 🍽️

A restaurant sim demo game written in rust. Found here on Github. Can be compiled by yourself from source with 'cargo build --release' and can be found in the /target/release folder as an application file.
